# Exit - core_exit.sh

This script allows for the use of exit codes which will print different outputs to LinuxGSM logs. Running `core_exit.sh` defaults exitcode variable to 0 which stands for a proper exit

| Exit Mode | Exit Code |
|-----------|-----------|
| Normal    | 0         |
| FATAL     | 1         |
| ERROR     | 2         |
| WARN      | 3         |
